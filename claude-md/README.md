# CLAUDE.md Templates for Legacy Modernization

Three ready-to-use CLAUDE.md files. Drop into your repo, customize, commit.

- **`CLAUDE_TEMPLATE.md`** — clean template with placeholders. Start here.
- **`CLAUDE_EXAMPLE.md`** — worked example (fictional enterprise system) showing how a fully-populated CLAUDE.md should look.
- **`CLAUDE_petclinic.md`** — tailored for Spring PetClinic. Use with `demo/DEMO.md` for a 30-minute hands-on walkthrough.

See the [main README](../README.md) for the article that explains the philosophy behind these files.
